# jetty-detector
Arduino based project to detect movement in a jetty / pontoon.

### Aims
This project aims to utilise an arduino, SD Card shield and gyro to create a proof of concept to track motion of pontoons or jetty's.
### Hardware
- Arduino Uno
- MMA8451 Adafruit 6 axis gyro
- Unknown SD card shield for arduino
### Installation
Open `main/main.io` in arduino IDE. Select Compile and wait for comfirmation.
